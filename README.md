# COMP3011 Movies API

## Overview
This project is a RESTful web API for managing a movie catalogue. It allows clients (e.g., Postman or another app) to create, view, update, and delete movie records stored in an SQL database. The API returns JSON responses and uses standard HTTP methods and status codes.

## Features
- Movie resource with full CRUD operations (Create, Read, Update, Delete)
- Reviews/ratings linked to movies (one-to-many relationship)
- Analytics endpoints (e.g., top-rated movies, ratings by genre)
- Input validation and consistent error responses
- Authentication for protected actions (e.g., create/update/delete)
